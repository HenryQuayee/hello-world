Introduction

Welcome to the website design documentation! This page can be utilized in order to understand technical specifications of designing this website.

For starters, we will be using basic HTML and CSS to design this webpage. Recall using the html element to setup the webpage. In addition, we will be using the style element to style our website. .

  • Basic set up is crucial to unveil a functioning webpage
  • CSS styling will cause your page to become more functional, accessible, and user friendly

Prior Knowledge

In the prior lessons, you have learned the use of header body main footer elements and you should continue to implement these in your webpage.

We will continue to expand on your element toolkit throughout this web design process. Make sure you are recalling the use of classes in CSS and how to insert them in the HTML. For example: .white-text{ color: white; }


HTML

In this project, you are expected to have added these additional elements to your HTML

  • nav
  • a
  • p
  • section
  • ul
  • ol
  • li

These elements will allow for more versatility in your web design. You will be able to create a navigation bar with links, a more organized main element, and both unordered and ordered lists.


CSS

In addition to the extra HTML elements, the following style codes should be utilized as well.

  • background-color
  • font-size
  • font-family
  • display
  • padding
  • margin
  • position
  • height
  • width

These attributes will allow for your web page to be more accessible and easily navigated. Style is extremely important to keeping your web user engaged and allowing them to easily navigate and comprehend the information you have provided.
